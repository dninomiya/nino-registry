たとえば以下のような依頼が通るようになります。

- ninoui のコンポーネントリストを教えて
- ninoui の input-image を使ったアバターフォームを作って

## 接続方法

1. プロジェクト環境で shadcn MCP を有効にする。(Claude Code, Cursor などに対応)

```bash
pnpx shadcn@latest mcp init
```

2. プロジェクトにレジストリを登録する

`components.json` にレジストリを登録する。`{name}` のままでOKです。

```json
{
  "registries": {
    "ninoui": "https://registry.dninomiya.com/r/{name}.json"
  }
}
```

以上でセットアップは完了です。たとえば以下のようにエージェントに依頼することができます。

- `ninoui` のすべてのコンポーネントを見せて
- `ninoui` の `input-image` を使ったアバターフォームを作って

MCPが反応しない場合、エディターを再起動します。
