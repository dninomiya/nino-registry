## 特徴

- **Next.js ベース**: App Router を使用した最新の Next.js アプリケーション
- **shadcn/ui ベース**: カスタマイズ可能でアクセシブルなコンポーネント
- **Tailwind CSS 4**: ユーティリティファーストの CSS フレームワーク
- **TypeScript**: 型安全な開発環境
- **フルカスタマイズ可能**: コンポーネントのコードを直接プロジェクトに追加するため、完全なカスタマイズが可能

## インストール

nino/registry は、パッケージとしてインストールするのではなく、必要なコンポーネントのコードを直接プロジェクトに追加します。これにより、以下の利点があります：

- コンポーネントの完全なカスタマイズが可能
- 依存関係の最小化
- プロジェクト固有の要件に合わせた調整が容易

コンポーネントを追加するには、各コンポーネントのページに記載されているコマンドを実行します。
