## インストール

```bash
# ページ全体
pnpx shadcn@latest add https://registry.dninomiya.com/r/marketing-page.json

# ヒーローセクション
pnpx shadcn@latest add https://registry.dninomiya.com/r/hero-section.json

# 機能セクション
pnpx shadcn@latest add https://registry.dninomiya.com/r/features-section.json

# 料金プランセクション
pnpx shadcn@latest add https://registry.dninomiya.com/r/pricing-section.json

# 利用者の声セクション
pnpx shadcn@latest add https://registry.dninomiya.com/r/testimonials-section.json

# よくある質問セクション
pnpx shadcn@latest add https://registry.dninomiya.com/r/faq-section.json

# CTAセクション
pnpx shadcn@latest add https://registry.dninomiya.com/r/cta-section.json

# ロゴセクション
pnpx shadcn@latest add https://registry.dninomiya.com/r/logos-section.json

# ヘッダー
pnpx shadcn@latest add https://registry.dninomiya.com/r/header.json

# フッター
pnpx shadcn@latest add https://registry.dninomiya.com/r/footer.json
```
