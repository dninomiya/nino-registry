## インストール

```bash
pnpx shadcn@latest add https://registry.dninomiya.com/r/preview-link.json
```

## 使用法

```tsx
import { PreviewLink } from "@/registry/new-york/preview-link";

export default function Page() {
  return <PreviewLink url="https://example.com" />;
}
```

## 例

### プレビューが取得できないリンク
